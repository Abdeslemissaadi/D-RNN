# 🧠 DSA — Deep Sentiment Architecture (Reproduction IEEE 2023)

> Reproduction et amélioration d'un article scientifique publié sur IEEE (2023).
> Architecture BERT-large-cased + D-RNN + ABSA + PBSA pour l'analyse de sentiment multi-dataset.

---

## Description

Ce projet reproduit l'architecture **DSA (Deep Sentiment Architecture)** décrite dans un article IEEE 2023,
en visant les mêmes résultats — voire de meilleures performances — grâce à un algorithme personnalisé
combinant deep learning, embeddings contextuels et analyse lexicale pondérée.

L'analyse de sentiment est réalisée sur **3 datasets distincts**, en classification 3 classes :
`negative / neutral / positive`
